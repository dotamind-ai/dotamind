# Dotamind AI — Technical Specification

## 1. Описание проекта

Dotamind AI — сервис анализа матчей Dota 2 с использованием искусственного интеллекта.

Пользователь вводит Match ID своей игры и получает подробный разбор.

---

## 2. Главная задача

Помочь игрокам понимать:

- почему они проиграли;
- какие ошибки совершили;
- как улучшить свою игру;
- какие решения были лучше.

---

## 3. Первый релиз (MVP)

Версия 0.0.1 должна уметь:

1. Получить данные матча по Match ID.
2. Определить:
   - героя;
   - роль;
   - KDA;
   - GPM;
   - XPM;
   - Last Hits;
   - предметы;
   - способности.

3. Создать базовый отчет:

- главные ошибки;
- проблемы с фармом;
- проблемы с предметами;
- оценка игры.

---

## 4. Будущие функции

- Анализ драфта.
- Советы по выбору героя.
- Анализ прокачки навыков.
- Оптимальный маршрут фарма.
- Анализ командных сражений.
- Персональный AI тренер.
- История прогресса игрока.

---

## 5. Монетизация

Бесплатная версия:

- ограниченное количество анализов.

PRO версия:

- полный анализ;
- история матчей;
- персональные рекомендации;
- глубокая статистика.

---
