# Dotamind Telegram Bot

## Цель

Создать Telegram-бота для анализа матчей Dota 2.

---

## Команды

/start

Запуск бота.


/analyze MATCH_ID

Анализ игры.

---

## Первый функционал

1. Получить Match ID.

2. Запросить данные матча.

3. Найти игрока.

4. Получить:
- героя;
- KDA;
- GPM;
- XPM;
- предметы.

5. Создать отчет.

---

## Будущие функции

- AI объяснение ошибок.
- Анализ драфта.
- Советы по пикам.
- Анализ леса.
- Персональный тренер.
